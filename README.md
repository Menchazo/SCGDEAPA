# Proyecto de Gestión de Adultos Mayores

Esta aplicación es una solución tecnológica diseñada para la gestión de datos de los adultos mayores en el Consejo Comunal Territorio Social Pantano Abajo I.

## Objetivo del Proyecto

El principal objetivo de este proyecto es optimizar los procesos de seguimiento y gestión de datos, buscando asegurar un manejo adecuado de la información y, en última instancia, mejorar la calidad de vida de nuestra valiosa comunidad de adultos mayores.

## Desarrollado por

Este proyecto ha sido desarrollado por **Gladys Morles** y **Juliangie Ventura**, estudiantes del Programa Nacional de Formación (PNF) en Informática de la Universidad Politécnica Territorial de Falcón "Alonso Gamero" (UPTAG).

## Tecnologías Utilizadas

Este proyecto fue construido utilizando las siguientes tecnologías:

*   **Vite**: Un entorno de desarrollo frontend moderno y rápido.
*   **React**: Una biblioteca de JavaScript para construir interfaces de usuario.
*   **Tailwind CSS**: Un framework de CSS para un diseño rápido y personalizado.
